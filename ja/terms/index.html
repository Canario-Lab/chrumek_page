「Chrumek」アプリ利用規約

発効日: 2026年2月8日
バージョン: 1.0

1. 提供者

  1. 「Chrumek」アプリ（以下「本アプリ」）の提供者は以下のとおりです。
    Canario-Lab / Chrumek App
    メール: chrumek@chrumek.app
    （以下「提供者」）
  2. 本規約は、本アプリの無料機能および有料機能の利用条件を定めるものです。

2. 定義

本規約において、以下の語は次の意味で用います。

  1. ユーザー: Apple 端末上で本アプリを利用する者。
  2. 記録: 購入を見送ったこと、より安い選択肢を選んだこと、サブスクリプションを解約したことなど、ユーザーが本アプリに入力する節約関連の情報。
  3. 「貯金に移しました」: 記録した金額を実際に取り分けたことをアプリ上で確定する操作。未確定の一覧から履歴へ移り、実際に貯金した金額として扱われます。
  4. Premium: サブスクリプションによって提供される本アプリの有料機能。
  5. 無料トライアル: App Store が対象ユーザーに提供する、Premium の無料試用期間。

3. 技術要件

  1. 本アプリの利用には、対応する Apple 端末と、本アプリが対応する iOS バージョンが必要です。
  2. 一部の機能（特にサブスクリプションの購入や復元）には、インターネット接続と Apple ID が必要です。

4. アプリの性質

  1. 本アプリは、使わずに済んだお金を記録し、実際に貯金へ移したことを確認することで、貯金習慣をサポートするものです。
  2. 本アプリは金融サービス、銀行、投資助言サービスではありません。実際の送金を代行したり、取引の仲介をしたりするものでもありません。
  3. 提供者は、ユーザーが特定の貯金成果を達成することを保証しません。

5. データとプライバシー

  1. 本アプリではアカウント登録は不要です。
  2. ユーザーが入力したデータは、原則としてユーザー端末内にローカル保存されます。
  3. データ処理の詳細は、プライバシーポリシーに記載します。

6. 無料機能と回数制限

  1. 本アプリには無料機能と Premium 機能があります。
  2. Premium を有効にしていないユーザーは、実際の貯金として確定できる回数が最大10回まで無料です。
  3. 10回の無料回数を使い切った後は、さらに確定するには有効な Premium が必要です（対象者に無料トライアルがある場合はそれを含みます）。
  4. 無料回数のカウンターは:
  • 実際に確定したときのみ増えます。
  • 無料トライアル終了後や Premium 失効後でも減らず、リセットもされません。

7. Premium、サブスクリプション、支払い

  1. Premium は、Apple（App Store）が処理する自動更新サブスクリプションとして提供されます。
  2. 価格、請求周期、利用可能なプランは、アプリ内の購入画面に表示され、App Store の設定に従います。
  3. 購入確定後、料金はユーザーの Apple ID に請求されます。
  4. 現在の請求期間が終了する前に Apple ID 設定で自動更新をオフにしない限り、サブスクリプションは自動更新されます。
  5. 提供者は、カード情報などの支払いデータを処理しません。支払い処理は Apple が行います。

8. 無料トライアル

  1. 本アプリは、対象ユーザーに対して Premium の無料トライアル（例: 14日間）を提供する場合があります。
  2. 無料トライアルは、Apple の条件や過去の利用履歴により、すべてのユーザーに提供されるとは限りません。
  3. トライアル終了後に継続を希望しない場合、ユーザーは Apple のルールに従って Apple ID 設定からサブスクリプションを解約する必要があります。
  4. 無料トライアルを終了または解約しても、無料回数のカウンターはリセットされません。

9. RevenueCat

  1. 本アプリは、Premium 状態の確認やオファー表示など、サブスクリプション管理の技術基盤として RevenueCat を利用する場合があります。
  2. 購入処理そのものは Apple が行い、RevenueCat はアプリ内での購読状態管理を補助します。
  3. この範囲におけるデータ処理の詳細は、プライバシーポリシーに記載します。

10. 購入の復元

  1. ユーザーは、Apple の仕組みおよび本アプリの「購入を復元」機能（該当バージョンで提供される場合）を使って、過去のサブスクリプションを復元できます。

11. サポートと不具合報告

  1. 技術的な問題は、chrumek@chrumek.app まで報告できます。
  2. 「バグを報告」機能がある場合、本アプリは診断情報を含むメール下書きを作成することがあります。送信はユーザー自身の確認後にのみ行われます。
