プライバシーポリシー

1) データ管理者

本アプリのデータ管理者は以下のとおりです。
Canario-Lab / Chrumek App
連絡先メール: chrumek@chrumek.app

2) 対象と目的

Chrumek は、買わずに済んだ支出や、実際に貯金へ移した金額を記録することで、貯金習慣づくりを助けるアプリです。

本アプリではユーザー登録は不要で、原則として利用者を直接特定できる個人データは収集しません。

3) アプリ内で扱うデータ（端末内にローカル保存）

アプリに入力したデータ（記録、金額、日付、カテゴリ、目標、各種設定など）は、端末内にローカル保存されます。

これらのデータが自動的に当社や当社サーバーへ送信されることはありません。

4) システム権限を使う機能

アプリは、動作に必要な範囲でのみシステム機能や関連データを利用します。

  • システム通知: リマインダーを有効にすると、端末のOS（例: iOS または Android）が設定に従って通知を配信します。
  • 言語 / 地域 / 通貨 / テーマ: アプリ内の表示や書式を整えるためにのみ使われ、当社には送信されません。

5) バグ報告 / サポート連絡

「バグを報告」機能を使うか、サポートへ連絡する場合:

  • アプリはメールアプリ内で下書きを作成することがあります。
  • 下書きには診断情報（アプリのバージョン、端末のOS、言語 / 通貨設定など）が含まれることがあります。
  • メールは、利用者が自分で送信を確定した場合にのみ送られます。

送信されたメールは、問い合わせ対応のために、利用者のメールサービス提供者および当社によって処理されます。

6) 購入とサブスクリプション（Apple / Google + RevenueCat）

アプリで有料機能（サブスクリプション）を提供する場合:

  • Apple端末では、決済は Apple（App Store / In-App Purchases / StoreKit）が処理します。
  • Android端末では、決済は Google（Google Play / Google Play Billing）が処理します。
  • アプリは、オファー表示・購読状態の確認・Premium 権限の付与など、サブスクリプション管理の技術基盤として RevenueCat を利用します。

サブスクリプション処理の過程で、以下のような技術データが扱われる場合があります。

  • 購入 / 取引 ID と商品情報
  • サブスクリプションの状態（有効 / 無効、更新状況）
  • 購読状態を紐づけるための技術的識別子（例: SDK が生成する匿名の App User ID）

RevenueCat はサブスクリプション管理機能を提供するためにこれらのデータを扱い、Apple または Google はそれぞれのポリシーおよびプライバシーポリシーに基づいて決済関連データを処理します。

7) 解析とトラッキング

本アプリは、ユーザー追跡のための解析ツールや広告 SDK を使用せず、データを販売することもありません。

8) ウェブサイト

当社ウェブサイトにアクセスした場合、ホスティング事業者（例: GitHub Pages）が、セキュリティ・診断・サービス維持の目的で、IP アドレス、ブラウザ種別、アクセス日時などの標準的な技術ログを自動的に処理することがあります。処理の範囲やルールは、ホスティング事業者のポリシーによります。

9) データの受領者

データが開示される可能性があるのは、次の場合に限られます。

  • Apple、Google、および RevenueCat（購入・サブスクリプション処理の範囲）
  • 法令に基づき権限を持つ公的機関

10) 保存期間

  • アプリに入力したデータは、利用者が削除するまで端末内に保存されます。
  • メールのやり取りに含まれるデータは、問い合わせ対応および必要な権利保護のために必要な期間保存されます。

11) 利用者の権利（EU / EEA）

具体的な状況で当社が利用者の個人データを処理する場合（例: メールで問い合わせを受けた場合）、利用者には次の権利があります。

  • 開示請求、訂正、削除、処理制限
  • 処理への異議申立て
  • 管轄監督機関への苦情申立て

情報提供義務の範囲は、とくに GDPR 第13条に基づきます。

12) セキュリティ

当社は、サービスの性質に応じた合理的な技術的・組織的安全対策を講じます。アプリのデータは端末内に保存されるため、端末自体を保護すること（例: パスコード、パスワード、PIN、生体認証 など）も重要です。

13) ポリシーの変更

本ポリシーは、アプリ機能や法的要件の変更に応じて更新される場合があります。最新版はアプリのウェブサイトで公開します。
